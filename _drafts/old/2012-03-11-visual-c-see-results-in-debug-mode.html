MS Visual Studio terminates program in Debug mode immediately after executing it,
so it is impossible to see output of the program.

Solution:

#ifdef _DEBUG
    // enable seeing the results in debug mode 
    // require #include 
    int l;
    std::cin>>l;
    std::cout<< "Press any key (debug mode) to terminate the program"<<td::endl;
#endif
