How to install Sun Java on Ubuntu?


Guide
Extracted commands

  1. Download and extract NOT RPM sefl extracting file I run it in /opt directory

  2. Install it to system 

        sudo update-alternatives --install "/usr/bin/java" "java" "/opt/jre1.6.0_30/bin/java" 1

  3. Make it default 

        sudo update-alternatives --set "java" "/opt/jre1.6.0_30/bin/java"



How to increase memory for JVM? 



java -Xmx512m -jar jarExample.jar

This launches JVM with maximum memory 512MB for an axecution of a jar file
