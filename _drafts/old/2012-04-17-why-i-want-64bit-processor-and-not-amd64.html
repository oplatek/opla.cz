  • 64 bit processors compute not faster, because the pointers are bigger
    and less data in registers, caches etc.



  • We want 64 bit processor because we need bigger VAS (Virtual address space)


  • In 64 bit VAS can live all processes so it can be implemented
    SINGLE ADDRESS SPACE for all processes. So it is easy to
    access     memory mapped files e.g. like Oracle does.
    AMD does not support it!
