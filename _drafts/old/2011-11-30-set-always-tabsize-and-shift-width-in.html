--- layout: post title: Set always tabsize and shift width in Vim together date: '2011-11-30T15:40:00.000+01:00' author: Ondrej Platek tags: - Vim - Techtips modified_time: '2012-07-23T17:17:21.115+02:00' blogger_id: tag:blogger.com,1999:blog-1883852367990943689.post-562963725484739339 blogger_orig_url: http://oplatek.blogspot.com/2011/11/set-always-tabsize-and-shift-width-in.html --- Lessons learned: Shift width(sw)and tabulator size (ts) are two different variables in Vim.
Shift width is used for shifting blogs via < and > keys.
Tabulator size is used when you press Tab in insert mode.
Two different things, but really useful to have them set up with the same value. In order to used them interchangeably.
So put following in ~/.vimrc:

set sw=4
set ts=4
