In the guide-default wiki you are advised to do https clone: 

https://bitbucket.org/username/project-name.git/wiki
Then git push origin master does not work as you probably wanted and ask from obvious reason for password and login. However if you clone via ssh it works pushing works well. So first do. 

git clone git@bitbucket.org:UserName/project-name.git/wiki
git push origin master # will use your ssh key
