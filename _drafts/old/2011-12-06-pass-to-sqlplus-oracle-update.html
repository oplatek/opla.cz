Problem


Inserting '&' to sqlplus failed. See results below:

create table test3 ( c varchar(30));
insert into test3 values ('blabla & testvalue');
Enter value for testvalue:


Solution is to turn off interpreting string after & like variable

set define off;




If you need variables it is necessary to define escape character.

set define on
set escape on

See discussion at stackoverflow.com!
